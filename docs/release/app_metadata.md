# Store Metadata Draft

## App Name
CaloriesApp

## Tagline
Log meals in seconds with AI and stay on track.

## Short Description
Camera-first nutrition tracking with editable AI analysis.

## Long Description
CaloriesApp helps you log meals from photos, estimate calories and macros, and track progress with clear reports. All analysis is approximate and user-editable.
