# Design System

## Tokens
- Color primary: `#2F6B5F`
- Surface: `#F7FAF8`
- Text primary: `#102018`
- Error: `#B42318`

## Typography
- Display: 32/40
- Headline: 24/32
- Title: 18/24
- Body: 16/24
- Caption: 13/18

## Spacing
- 4, 8, 12, 16, 24, 32

## Radius
- 8 (inputs), 12 (cards), 20 (modals)

## Components
- Buttons: primary, secondary, text, destructive
- Inputs: text, numeric, segmented, search
- Cards: summary, meal item, chart, recommendation
- Charts: light grid, clear labels, calm palette

## Accessibility
- Touch targets >= 44dp
- WCAG contrast baseline
- Dynamic text scaling support
