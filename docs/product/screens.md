# Screen Specifications

## 1. Splash/Boot
- Purpose: initialize session, locale, remote config.
- Layout: logo, progress indicator.
- Loading: full-screen spinner.
- Error: retry + offline mode.

## 2. Onboarding
- Purpose: explain value + disclaimer.
- Components: hero text, benefit bullets, continue CTA.
- Validation: consent required.

## 3. Auth
- Purpose: login/register.
- Components: email/password fields, OAuth buttons.
- Error: invalid credentials, network timeout.

## 4. Profile Setup
- Fields: age, sex, height, weight, activity, preference.
- Validation: numeric ranges.

## 5. Goal Setup
- Fields: calorie target, optional macros.
- Auto: TDEE baseline suggestion.

## 6. Home Dashboard
- Sections: today summary, quick actions, trends snippet.
- Empty: prompt first meal log.

## 7-10. Capture/Preview/Analysis/Edit
- Camera capture and upload.
- Result card per ingredient with grams/confidence.
- Editable grams and ingredients before save.
- Low-confidence warning triggers mandatory review.

## 11. Food Diary
- Group by day and meal type.
- Actions: edit, delete, duplicate.

## 12. Reports
- Daily/weekly/monthly tabs.
- Chart card + macro split + goal deviation.

## 13. Recommendations
- Concise practical cards.
- Localized language output.

## 14. Barcode Scanning
- Scan screen + product confirmation.
- Fallback manual entry when not found.

## 15. Meal Planner
- Generate plan from goals/preferences.
- Save plan to diary option.

## 16. Weight Tracking
- Log weight + trend chart.

## 17-19. Settings/Language/Notifications
- Locale switcher, unit switch, calorie target update.
- Notification schedule controls.
