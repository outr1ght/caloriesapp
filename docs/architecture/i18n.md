# Internationalization Strategy

## Supported Locales
- `en`, `es`, `de`, `fr`, `ru`

## Mobile
- Flutter `gen_l10n` with ARB files.
- Auto locale detect on first launch.
- User override persisted in secure/local storage.

## Backend
- `Accept-Language` + user setting fallback.
- Localized error catalog with stable keys.
- Recommendation generation language forced by user locale.

## Units and Formatting
- User setting for metric/imperial.
- Locale-aware date/number formatting in UI.
- Nutrition units normalized in backend and converted for display.
